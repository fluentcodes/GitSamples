Sandbox css: Textarea Nowrap

Avoid textarea text wrapping with css/ scroll bars

When working on this topic for elasticobjects project to put some working examples on a page, I remembered why I stopped working as a frontend developer more than twenty years ago. Different behaviour on the browsers. I've choosen proposed "whitespace:no-break" first, which worked on chrome but not on firefox

Think that Firefox did it right, since whitespace includes new line and so "new line" should be ignored.

no entry

Default behaviour is breaking whitespaces when the text is too long:

Attribute wrap=off

The attribute solution works on all browsers: No break for whitespace if text is too long: But I don't like to put the attribute on every textarea.

white-space: pre;

This style sheet solution from stackoverflow works as wrap="off".

  • https://developer.mozilla.org/de/docs/Web/CSS/white-space
  • https://www.w3schools.com/cssref/pr_text_white-space.asp

Non working solutions

text-overflow: ellipsis
